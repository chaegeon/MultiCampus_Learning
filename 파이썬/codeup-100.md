#### ord() , chr()
- ord()
  - 특정한 한 문자를 아스키 코드 값으로 변환해주는 함수

- chr()
  - 아스키 코드값을 문자로 변환해주는 함수((10진수, 16진수 사용 가능)

- ord => print(ord('A')) => 65
- chr => print(chr(65)) => A
